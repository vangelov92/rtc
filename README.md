# RTC

A small app which allows P2P web chat using the WebRTC standard and the RTCPeerConnection API.

## Available Scripts

To build the client, cd into the client directory and run:

### `npm run build`

This builds the app's front-end code into a dist folder on the same level as the client.

Before starting the server, make sure the client has been built first, then cd into the server directory and run:

### `npm run serve`

See the section about [deployment](https://facebook.github.io/create-react-app/docs/deployment) for more information.
